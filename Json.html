JSON

JSON: JavaScript Object Notation.
JSON is a syntax for storing and exchanging data.
JSON is text, written with JavaScript object notation.

exchanging data

sending->Recieving->storing

sending-data

If you have data stored in a JavaScript object, you can convert the object into JSON, and send it to a server: JSON.stringify()
stringify make the object or any data type to text;

recieving-data

if you receive data in JSON format, you can convert it into a JavaScript object: JSON.parse()
parse() make text to javascript object or any data type
